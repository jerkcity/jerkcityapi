Base URL: https://www.jerkcity.com/

Fields only present if data is available: tags, dialog, hd, hifi

GET /tags/:tag/index.json
=========================

Example values for :tag: `dicky-bird`, `toilet`

Returns a list of episode IDs or HTTP code 404 if tag not found.

```javascript
{
   "episodes" : [
      6144
   ]
}
```
